Клас серіалізувати в JSON

1. Створити клас, в якого публічні поля будуть серіалізуватись.
2. Клас має реалізувати інтерфейс implements JsonSerializable
3. В класі потрібно реалізувати метод інтерфейсу JsonSerializable:
   public function jsonSerialize() {
           return [
               'поле1' => $this->поле1,
               'поле2' => $this->поле2,
               'поле3' => $this->поле3
           ];
       }
4. Якщо якесь поле є посиланням на інший клас (ствоений користувачем), тоді для цього класу
   потрібно окремо виконати пункти 1-3.

Використання: json_encode($this, JSON_UNESCAPED_UNICODE)
$this - якщо команда викликається в методі класу, інакше - змінна-об'єкт;
JSON_UNESCAPED_UNICODE - дозволяє правильно записати кириличний текст, оскільки в налаштуваннях
редактора в PHPStorm встановлено UTF-8 кодування.


Десериалізація з JSON

$dataNew = json_decode($dataJson), де $dataNew об'экт типу StdClass.
Тому в класі бажано створити метод, який подібний до конструктора і на основі об'экту типу StdClass
створює об'экт користувацького типу.
